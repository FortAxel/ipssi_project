# Storybook Kids App

A web application for children's story reading - CDA Training Project

## 📚 About

Educational platform allowing children to read illustrated stories page by page, 
with features like favorites, reading history, and text-to-speech integration.

**Target audience:** Parents and children  
**Language:** French (UI), English (code/docs)

## 🎯 Project Status

- ✅ **Milestone 1:** Functional Specification (January 2026)
- 🔄 **Milestone 2:** Methodology + UI/UX Design (February 2026)
- ⏳ **Milestone 3:** Database Modeling (March 2026)
- ⏳ **Milestone 4:** Technical Design + Development (April 2026)
- ⏳ **Milestone 5:** Beta + Tests + Security (May 2026)
- ⏳ **Milestone 6:** Final Release + Deployment (June 2026)

## 🛠 Tech Stack

**Backend:**
- PHP 8.2+
- Symfony (latest LTS)
- MySQL
- Doctrine ORM

**Frontend:**
- React 18
- TypeScript
- Responsive design (mobile-first)

**DevOps:**
- Docker & Docker Compose
- GitHub Actions (CI/CD)
- Docker Hub

**External APIs:**
- Text-to-Speech service (planned)

## 📋 Project Management

- **Methodology:** Kanban (adapted for solo development)
- **Board:** GitHub Projects
- **Milestones:** 6 monthly milestones
- **Branches:** 
  - `main` - Stable releases (milestone deliverables)
  - `develop` - Integration branch (daily work)
  - `feature/*` - Feature branches (optional)
